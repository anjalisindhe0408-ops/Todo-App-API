 # Todo App Backend API

A RESTful backend application built using Spring Boot for managing users and their todo tasks.

## Features
- User management APIs
- Todo management APIs (Create & Read)
- One-to-Many relationship between User and Todos
- RESTful architecture
- MySQL database integration

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## API Endpoints
- GET /api/users
- POST /api/users
- GET /api/todo
- POST /api/todo
